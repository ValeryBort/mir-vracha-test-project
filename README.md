### Автотест на прохождение регистрации под ролью студента на проекте Мир Врача

#### Как запустить Автотест

##### Технические требования

- Python 3.6
- [Poetry](https://python-poetry.org/docs/)

##### Настройка окружения

- Установить зависимости

        $ poetry update

- Запустить тест

        $ poetry run pytest
